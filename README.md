# Swerve Lib
an extension of the eclipselib, to add functions that allow for Swerve modules to be used. this library is adapted from many FRC elements, rewritten in rust (Original Repo: https://github.com/NoahLacks63/VEX-Differential-Swerve-Drive) 

## Wait, Swerve Modules??
Yes! Swerve modules... Team VURC BBR1 has begun the development on differential swerve modules, we believe that swerve modules in the right environment and given enough development, swerve modules can be viable. and not just viable but dominant
